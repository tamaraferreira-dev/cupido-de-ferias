# 🏹 Cupido de Férias

Projeto interativo e divertido desenvolvido para o Dia dos Namorados, focado em quem passou a data solteiro(a) e quer dar boas risadas.

## 🚀 Tecnologias Utilizadas
- HTML5 (Estrutura)
- CSS3 (Estilização e posicionamento absoluto)
- JavaScript Vanilla (Lógica do botão fujão e gerador de desculpas)

## 🧠 O que eu aprendi com este projeto:
- Como manipular elementos HTML usando o JavaScript (`document.getElementById`).
- Como escutar eventos do usuário como `click` e `mouseover`.
- Uso do objeto `Math` (`Math.random` e `Math.floor`) para criar lógicas aleatórias.
- Posicionamento relativo e absoluto no CSS.
